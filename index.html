int add(int a, int b) {
    return a + b
}

int mult(int a, int b) {
    return a * b;
}

Functional Programming Fundamentals

  • Functions are first class citizens
  • Pure functions
    • No side effects
    • Referential transparency
  • Functions are first class citizens
    • - zapisujemy je do zmiennej, zwracamy, komponujemy
  • Pure functions
    • - funkcja czysta to taka która nie ma efektów ubocznych - jej wynik zależy tylko od argumentów wejściowych - można wywołanie tej funkcji zamienić na wynik
